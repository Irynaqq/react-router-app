# React Router App

Невеликий React-додаток, створений для ознайомлення з основами маршрутизації на боці клієнта за допомогою **React Router**.

## Функціонал
- Навігація між сторінками без перезавантаження
- Маршрути: Home, About, Contact
- Активні посилання через `NavLink`
- Сторінка 404
- Вкладені маршрути (`/about/team`, `/about/history`)
- Кнопка "Назад" через `useNavigate`
- Динамічна генерація маршрутів з масиву
- Сучасний `createBrowserRouter` + `RouterProvider`

## Технології
- **Vite**
- **React**
- **React Router**

## Як запустити проект
npm install
npm run dev
